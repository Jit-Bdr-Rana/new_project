
## Getting Started

## Employee Management System demo

First, install packages
```
npm install
```
Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Third, run json server
```
npm run server
```

Forth, run tailwind
```
npm run tailwind
```
At last ,here you go
(http://localhost:3001)
